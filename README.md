En este repositorio encontrarás los códigos fuente (archivos `.json`) de los agentes inteligentes desarrollados para el proyecto.
¿Cómo usarlos?
Para probar estos agentes en tu propia cuenta de n8n:
1.  Selecciona el agente: Abre el archivo `.json` que te interesa en este repositorio.
2.  Copia el código: Puedes copiar todo el texto del archivo o descargarlo a tu computadora.
3.  Pega en n8n:
    Ve a tu lienzo de n8n (Workflow).
    Haz clic en el fondo y presiona `Ctrl + V` (Pegar).
